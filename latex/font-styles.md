# Font styles

- medium:
  - command: `\textmd{}`
  - switch command: `\mdseries`

- bold:
  - command: `\textbf{}`
  - switch command: `\bfseries`


- upright:
  - command: `\textup{}`
  - switch command: `\upshape`


- italic:
  - command: `\textit`
  - switch command: `\itshape`


- slanted:
  - command: `\textsl`
  - switch command: `\slshape`


- small caps:
  - command: `\textsc`
  - switch command: `scshape`
