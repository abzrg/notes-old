## Original

- cmd-o: open
- f5   : reload data
- cmd-s: save data
- cmd-f: search in the item list
- cmd-?: paraview guide
- cmd-z: undo
- cmd-y: redo
- cmd-b: camera undo
- cmd-g: camera redo

- alt-a: apply
- alt-r: rest
- alt-d: delete

- v : find data


## custom

- cmd-shift-k: customize shortcut:
- cmd-shift-r: reload data

- alt-ctrl-c: clip
- alt-ctrl-s: slice
- alt-ctrl-t: contour
